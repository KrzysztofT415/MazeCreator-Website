For now:

> Right-click-drag => moving grid

> Left-click-drag => drawing and erasing cells - depends on highlighted mode in top menu

> Mouse-wheel => zooming grid

* Click on Algorithm's title to change algorithm

There are 3: RecursiveBacktracking, Kruskal's and Prim's

* Buttons Draw/Erase change drawing mode

* Button Reset deletes all cells

* Button Visualize makes maze based on current algorithm

U can press it after generating one maze and changing algorithm to generate new one

U can press buttons' highlighted letters corresponding keys to make action
